### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 公告
5. 商品
6. 个人中心
7. 商品收藏
8. 商品订单
9. 购物车
10. 后台管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（操作方式管理、公告类型管理、会员等级类型管理、交易类型管理、游戏类型管理、投诉类型管理、游戏玩法管理）
4. 公告管理
5. 商品管理、商品评价管理、商品收藏管理、商品订单管理
6. 订单投诉管理
7. 用户管理
8. 轮播图信息
